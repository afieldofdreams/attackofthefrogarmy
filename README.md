# Attack of the Frog Army

A retro pixel-art action platformer — a ninja's last stand against an endless frog army.

## How to Play

Move your ninja through procedurally generated levels, fight waves of frogs, and defeat the boss at the end of each stage.

**Controls:**
- **Arrow keys / on-screen buttons** — Move left and right
- **Space / Jump button** — Jump
- **Z / Attack button** — Melee attack

Works on desktop and mobile (touch controls included).

## Running

Open `frog-army.html` in any browser — no build step or server required.
